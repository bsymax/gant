# 界面与配色：「极光指挥台 · Aurora v2」

> **0.2.0** 起，整体从「近黑底」调整为 **偏青灰的较亮深底 + 高对比文字 + 更明显的网格与描边**，在保留**科技感、轻度战略 HUD/游戏**质感的前提下，**减轻费眼、提高可读性**。

## 1. 设计目标

- **清晰优先**：主正文接近浅灰白 `#f0f4f8`，次要信息用 `#b8c5d4`、弱化用 `#7d8fa3`。
- **不丢气质**：顶光雾面（`radial-gradient`）、细网格、面板微内光、主按钮**青系渐变+描边**。
- **可维护**：**绝大多数颜色与组件壳子**在 **`web/app/globals.css`** 用 CSS 变量 + `.gant-*` 类统一，避免在几十个页面里手改色值。

## 2. 色板（`globals.css` 的 `:root`）

| 变量 | 作用 |
|------|------|
| `--gant-bg` / `--gant-bg-2` | 主背景、叠层，偏**蓝灰**（非纯黑 #000） |
| `--gant-fore` / `--gant-fore-secondary` / `--gant-muted` | 主字、次字、说明字 |
| `--gant-grid` / `--gant-line` / `--gant-line-soft` | 背景网格、面板边、内部分隔（**比 v1 更明显**） |
| `--gant-accent` | 高亮/焦点（天青） |
| `--gant-amber` | 与「战情/储备/上战场」等琥珀提示（不滥用） |
| `--gant-surface` / `--gant-input-bg` | 面板与输入区底色，**比旧版更亮** |

大改风格时：先调这几个变量，再微调 `.gant-panel` / `.gant-btn` 里的渐变与阴影。

## 3. 常用类（同一文件内）

- `.gant-panel`：大块内容容器
- `.gant-card`：储备区卡片
- `.gant-header-bar`：顶栏
- `.gant-input` / 继承同款边框的表单项
- `.gant-btn`：主按钮；`.gant-btn-ghost`：次要/退出
- `.gant-section-amber` / `.gant-section-cyan`：区标题强调色
- `.gant-text-muted` / `.gant-text-body` / `.gant-link`
- `.gant-gantt-bar`：时间轴上项目条
- `body` 使用 `gant-canvas` + 变量文字色（见 `app/layout.tsx`）

## 4. 若还要再调亮/调冷/调「更游戏化」

1. 只动 **`web/app/globals.css`**，**保存**后 `npm run dev` 下热更新即可看效果。  
2. 可微调：`--gant-bg` 再浅 1～2 个阶（例如从 `#1b2635` 到 `#1e2c3c`）观察对比。  
3. 更强「策略游戏感」可后续加：角落装饰线框、**不要**在每一页用不同色偏（统一走变量）。

## 5. 与产品文档的关系

PRD 里「智能化、科技化、游戏化」的**可落地入口**就是：**本文件 + `globals.css` + 指挥台/甘特等组件**。需求变更时若只改交互不改风格，以 **系统功能与文件地图** 里的文件为准。
